How to update to JEM 4 for Joomla 4 and PHP 8

Upgrading from a previous version to JEM 4 requires a procedure to follow. It's not as simple as 1,2,3, please read the following instructions carefully:

If you are on a clean JEM 2.3.5 for Joomla 3 (if you have template overrides for JEM, you must adapt them by yourself!).

Preparation

  • Make a backup!
  • Update to Joomla 3.10.11
  • Update to JEM 2.3.6 (Premigration version)
  • Make a new backup!

Step 1: Update Joomla to Version 4

During upgrade process, it's suggested to switch to Joomla 3 default template "Protostar" and disable all template overrides

  • Disable all JEM extensions, component, all JEM plugins, all JEM modules. DON'T uninstall!
  • To prepare the update from Joomla 3 to Joomla 4, go to Components > Joomla! Updates, under Options as the Update Channel choose Joomla Next
  • Save

You now see the Joomla 4 update information and "Pre-Update Check", which includes a check of all components and shows which of them are not ready to migrate. Perhaps you have to disable/uninstall other components/modules/plugins which are not yet ready for Joomla 4

  • Update your webspace to run under php 8+
  • Upgrade from Joomla 3 to Joomla 4 (choose the tab "Live Update" under "Joomla Update").

Step 2: Install JEM 4

Now you are in Joomla 4 (which looks different than in Joomla 3). Maybe you have to login again.

  • In the admin menu on the left go to System, then under Install choose Extensions
  • Install JEM 4, either with the web installer or by uploading the package you downloaded on www.joomlaeventmanager.net/download
  • Before switching to the frontend, change the positions of the JEM modules and the main menu module, to the positions of your new Joomla 4 template (e.g. the Joomla 4 default template "cassiopeia")
  • If you get a red error window, go to System > Global Configuration > Site in the box SEO and disable Use URL Rewriting
  • Enable all JEM 4 (component, modules, plugins)
  • Adapt the JEM settings (there are changes: e.g. date/time format)
  • Check if everything works fine!

Finished
