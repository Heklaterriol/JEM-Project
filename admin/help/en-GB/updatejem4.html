How to update to JEM 4 for Joomla 4 and PHP 8

  • In Joomla! 3 Update to Joomla! 3.10.11
  • Update to php 8.0+
  • Update to JEM 2.3.5
  • Export via ftp the folder media/com_jem/attachments with subfolders and attachments
  • Export via ftp the folder images/jem with subfolders and images
  • If you want the same JEM settings, you have to write it down.
  • Export via JEM export all the events (choose categorycolumn=yes, select all categories und a startdate and enddate with all events inside)
  • Export categories, venues and cat_events
  • For safety reasons it would be better not to delete your old Joomla3! Better to install Joomla!4 parallele to old Joomla! 3
  • Install Joomla! 4
  • Install JEM 4
  • Go to JEM settings and set them like before or like you want
  • Import via ftp the folder images/jem with subfolders and images
  • Import via ftp the folder media/attachments with subfolders and attachments
  • Import via JEM import all the venues, categories, events and cat_events (exported csv files)
  • You have to adapt the user IDs (best in DB) - your admin ID in Joomla!4 will not be the same.
  • Finish
